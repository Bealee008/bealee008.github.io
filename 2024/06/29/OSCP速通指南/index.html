OSCP速通指南

Bealee
2024-06-29
OSCP

前言

由于目前大环境情况不乐观，有必要考些证书充实自己，那么应该去选择考哪些安全证书呢？目前国际上比较认可的是CISSP、OSCP证书，国内比较认可的是CISP-PTE这一类的证书，不过都是一些基础的证书，我个人认为OSCP证书比较好考一点在国际上认可度也比较高，如果白天要工作，业余花时间去准备OSCP，空余时间不多，所以最好花两个月左右时间去准备Offensive Security 的OSCP考试。在准备Offensive Security 的OSCP考试中我建议从打相应的靶场开始，合理规划两个月里的打靶时间，以下是我要分享的在准备Offensive Security 的OSCP考试需要知晓的事。

OSCP简述

OSCP（Offensive Security Certified Professional）是由Offensive Security（该公司开发和维护着Kali Linux，以下简称Offsec）推出的顶级信息安全认证。OSCP以其完善的体系和高难度著称，特别适合有一定基础但缺乏3年以上渗透工作经验的人员。认证课程结合了理论知识和实操训练，尤其注重实际操作能力。作为业界最知名的安全证书，OSCP要求考生在真实环境中进行渗透测试并提交详尽的报告。如果你对网络、主机渗透以及红队活动感兴趣，OSCP是你进入这一领域的理想起点。

OSCP考试规则

🔗OSCP官方考试规则

考试有三台独立靶机和一套域环境（三台机器），考试时长为24小时（考官24小时远程不间断监考）。

  • 三台独立机器，每台20分。
  • 域环境40分，需获取完整攻击链路才有40分。
  • 完成OSCP课程材料中 80% 的挑战练习并从挑战实验室获得30个“proof.txt”文件，可以获得10分。

考试合格分数为70分。

（Tips：也就是说，要通过考试，您的主要目标是掌握 Active Directory 40分 和至少一台独立机器 20分 + 奖励分 10分）

收费标准

🔗OSCP官方报名

目前官方收费有三个套餐：

  • 1649美元，包括课程、90天实验室和一次考试机会。
  • 2499美元，包括课程、一年实验室时间，两次考试机会、一年PG实验室、KLCP考试机会和OSWP考试机会。
  • 5499美元，包括一年实验时间，offensive官方所有证书的无限制考试尝试，不过有考试冷静期，7天后才能继续报重考。

OSCP前置知识

  • 扎实的 Linux 基础，对 Linux 文件系统，命令行，Shell Scripting 及各种应用（SSH，Cron Job，Vim， Netcat 等）都有深入的了解。这是后期 Linux 提权的基础。

  • 熟悉 Windows 系统命令行（CMD及Powershell）操作。这是后期 Windows 提权的基础。

  • 对于 TCP/IP 和网络有一定的理解。这能帮助大家排除在考试中的各种非预期异常。

  • 一定的编程经验，可以帮助检查公开exp当中的错误并且进行修改。

OSCP靶场和资源推荐

OSCP备考靶场：

​ 🤖Vulnhub：这个靶场的团队被Offsec收购现在为Offsec做靶场，免费，缺点就是被Offensive Security收购后，22年就停更了。

​ 🤖HTB：这里面机器质量很不错，并且设计的知识点比较多，一个月14美元。

​ 🤖PG：官方练习靶场，19美元一个月。

​ 🤖tryhackme：丰富的实验环境、社区支持，大部分免费，高级靶场约10美元/月。适合初学者和中级学习者。

​ 🤖hackmyvm：多可查看用户提交的WP，免费，靶机的质量还可以。

​ 🤖vulnyx：全部靶机只由一个作者出的（思路有局限性），免费，适合所有技能水平用户。

注意事项

  • 报名需要英文证件和美元信用卡，去公安局办个护照，再去银行申请一张人民币信用卡加办visa信用卡，报名也比较方便。
  • 考试前需要熟悉相关工具和命令，因为很多自动化的工具规定不能使用。
  • 提交报告时一定要注重报告质量，报告内容要体现思路清晰，不然会扣报告分。
  • 考试时间充足，规划好休息时间。
